First, what do these agenticAI system look like, and what might you find here?


1. agenticAI consists of collections of skills, agent descriptions, and workflows
2. agenticAI operates by interacting with an LLM to use skills and software tools
3. agenticAI is orchestrated by an interactive human terminal or closed-loop OS

And so, agenticAI system may consist of everything from: 

1. sophisticated text files which describe how agenticAI should think how to work
2. software APIs called MCPs which are hardware-run traditional software tools
3. LLM configurations and hardware for running LLMs in various environments
4. low-level software to ensure operations integrity and goal-oriented closed-loops
5. high-level application development on top of the capabilities and outputs of AI

For instance, in the last week (January 2026) we developed prototypes to do: 
1. a skills file to emulate a Pepperdine Law Grad deeply research a topic to PDF
2. a traditional OCR software system to give agenticAI access to image processing
3. a logging system at low-level to ensure all interactions are properly captured
4. a subagent to translate tough learning concepts into easier read content
5. a modern web-applet for generating slide decks from our research data
6. an email and notification system to operate underneath agenticAI systems
7. a subagent closed-loop monitoring system to repeat an agenticAI toward goals

What is OpenCode?

OpenCode is one of several agenticAI frameworks (c.f. ClaudeCode, Cursor, etc.) which 
controls some small measure of closed-loop feedback between an immediate user request
and the multiple executions of an LLM with tools to produce outcomes until a next input moment throughout a session. Each
of these frameworks follows more-or-less the same structure of skills, agents, tools, LLMs, and APIs, 
meaning that building on one more-or-less transfers across other agenticAI frameworks without lock-in, which is the
point here. OpenCode offers one of the only totally free platforms (despite negative impact investment from YC and LinkedIn) 
and provides introductory access to LLMs for free which are above the consumer class of house LLMs, while being swap-able into 
each of the open source LLM and commercial class LLMs, such as Anthropic, Gemini, OpenAI, etc. This makes 
OpenCode one of the only starting pads for serious agenticAI development outside of direct involvement in a specific vendor ecosystem, too.

We are using OpenCode (for now) despite: 
1. disagreeable availability of answers and staff to basic user requirements for enterprise approved development (c.f. [logging](https://github.com/anomalyco/opencode/issues/10886), and others)

3. genuinely insane LLM user agreements to "hold on to your data for as long as necessary"
4. their commercial entity status as a company invested in my YC (c.f. OpenAI) and LinkedIn (c.f. Microsoft)
